# Les Vecteurs

## 1. Définition
Un vecteur représente un déplacement. Il est caractérisé par :
- Une direction (la droite qui le porte)
- Un sens (de l'origine vers l'extrémité)
- Une norme (sa longueur)

## 2. Coordonnées d'un vecteur
Dans un repère $(O, \vec{i}, \vec{j})$, si $A(x_A, y_A)$ et $B(x_B, y_B)$ :
$\vec{AB} \begin{pmatrix} x_B - x_A \\ y_B - y_A \end{pmatrix}$

## 3. Somme de vecteurs (Relation de Chasles)
Pour tous points A, B, C :
$\vec{AB} + \vec{BC} = \vec{AC}$
