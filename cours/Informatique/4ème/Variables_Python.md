# Les Variables en Python

## 1. Qu'est-ce qu'une variable ?
Une variable est comme une "boîte" dans laquelle on peut stocker une information.

En Python, on crée une variable en utilisant le signe `=`.

```python
nom = "Alice"
age = 14
print(nom, "a", age, "ans.")
```

## 2. Les types de variables
Il existe plusieurs types de variables :
* **Entiers (int)** : Les nombres entiers comme `42`
* **Flottants (float)** : Les nombres à virgule comme `3.14`
* **Chaînes de caractères (str)** : Le texte comme `"Bonjour"`
