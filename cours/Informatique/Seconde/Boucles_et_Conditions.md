# Les Boucles et Conditions

## 1. La condition If
L'instruction `if` permet d'exécuter du code seulement si une condition est vraie.

```python
note = 15
if note >= 10:
    print("Vous avez la moyenne !")
else:
    print("Vous n'avez pas la moyenne...")
```

## 2. La boucle For
La boucle `for` permet de répéter une action un nombre de fois défini.

```python
for i in range(5):
    print("Ceci est le tour numéro", i)
```
