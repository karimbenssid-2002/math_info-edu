# Listes et Dictionnaires

## 1. Les listes
Une liste est une structure de données qui permet de stocker plusieurs éléments dans un ordre précis.

```python
fruits = ["Pomme", "Banane", "Cerise"]
print(fruits[0]) # Affiche Pomme
fruits.append("Orange") # Ajoute à la liste
```

## 2. Les dictionnaires
Un dictionnaire associe une valeur à une clé, au lieu d'un index.

```python
scores = {"Alice": 15, "Bob": 12}
print(scores["Alice"]) # Affiche 15
scores["Charlie"] = 18 # Ajoute au dictionnaire
```
